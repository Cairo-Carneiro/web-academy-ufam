O CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é uma linguagem de estilos utilizada para definir a apresentação visual de documentos estruturados, como HTML, XHTML ou XML. Enquanto o HTML é responsável por estruturar o conteúdo (títulos, parágrafos, imagens), o CSS cuida da aparência: cores, fontes, layout, espaçamentos e responsividade.
